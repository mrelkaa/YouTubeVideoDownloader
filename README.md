# YouTube Video Downloader

Небольшое приложение на Python для скачивания видео с YouTube через приятный фиксированный интерфейс.

## Возможности

- поле для ссылки;
- загрузка реальных доступных разрешений для конкретного видео;
- выбор формата `mp4`, `webm`, `m4a`, `mp3`;
- выбор папки сохранения;
- окно не растягивается.

## Установка

```powershell
python -m pip install -r requirements.txt
```

## Запуск

```powershell
python app.py
```

## Важно

- Для части разрешений выше `720p` нужен `ffmpeg`, иначе YouTube может не дать объединить видео и аудио.
- Для `mp3` и `m4a` тоже обычно нужен `ffmpeg`.
